C# programmer starter med å inkludere noe som kalles namespace (using System;)

Funksjonen til denne linjen er å gjøre det mulig for maskinen å skjønne kommandoene vi skal bruke.

 

I eksempel under brukes kommandoen console med parameteren writeline.

(Console.WriteLine("Hallo verden!");)

 

For at maskinen skal forstå denne kommandoen må vi inkludere et namespace som forteller maskinen hvordan den skal forstå denne kommandoen. For console kommandoen heter namspace system, derav kommandoen (using System;)

https://docs.microsoft.com/en-us/dotnet/api/system.console.writeline?view=netcore-3.1

https://docs.microsoft.com/en-us/dotnet/api/system?view=netcore-3.1

 

Deretter må vi deklarere et eget namespace slik at maskinen kan forstå de navnene vi bruker. I eksempelet under har jeg kalt namespace for Hallo_verden.

 

Et namespace brukes for å holde sett med names atskilt fra andre. Class names deklarert i ett namespace kommer ikke i konflikt hvis vi bruker de samme Class names i mer enn et namespace.

 

Et program i c# kan ha flere namespace som er laget av andre og vi kan ha flere egenlagde namespace.

Deretter lager vi en class. Det er som i css, vi oppretter klasser for å kunne lage objekter med egenskaper i HTML. På samme måte som i HTML CSS kan vi lage flere objekter fra en klassebeskrivelse.

 

namespace N1       // N1

{

    class C1       // N1.C1

    {

        class C2   // N1.C1.C2

        {

        }

    }

    namespace N2   // N1.N2

    {

        class C2   // N1.N2.C2

        {

        }

    }

}

 

I eksempelet bruker vi og en class, og det er fordi Main kallet alltid må stå i en class (eller struct)

 

Eksempel:

using System;

 

namespace Hallo_verden

{

    class Program

    {

        static void Main(string[] args)

        {

            Console.WriteLine("Hallo verden!");

        }

    }

Operatorer

søndag 27. september 2020

10:41

Aritmetiske:

+      Legger sammen to verdier.

-     Trekker en verdi fra en annen

*     Multipliserer en verdi med en annen

/     Dividerer en verdi med en annen

%     Finner rest etter heltallsdivisjon det vil si 7,5%2=3 hele og 1,5 i rest. 7.5%2=1.5

++    Legger til 1 på et heltall. (tall++ er det samme som tall=tall+1)

-    Trekker fra 1 på et heltall. (tall-- er det samme som tall=tall-1)

For å opphøye en verdi i c# bruker vi Math.Pow

 

Sammenligne

==    Hvis to verdier er like, true.

!=    Hvis to verdier er ulike, true.

>    Hvis verdi til venstre er større enn verdien til høyre, true.   

<    Hvis verdi til venstre er mindre enn verdien til høyre, true.

> =    Hvis verdi til venstre er større enn eller lik verdien til høyre, true.

<=    Hvis verdi til venstre er mindre enn eller lik verdien til høyre, true.

 

Tilordne

=    Variabel får verdi av aritmetisk beregning.

Eksempel:    sum = tall1 + tall2

+=     Variabel får verdi summen variabelen hadde fra før + verdi.

Eksempel:    sum += tall (sum=sum + tall)

-=     Variabel får verdi summen variabelen hadde fra før - verdi

Eksempel:    sum += tall (sum=sum - tall)

*=     Variabel får verdi summen variabelen hadde fra før multiplisert med verdi

Eksempel:    sum *= tall (sum=sum * tall)

/=     Variabel får verdi summen variabelen hadde fra før dividert med verdi

Eksempel:    sum /= tall (sum=sum / tall)

%=    Variabel får verdi restsum av variabelen delt med verdi

Eksempel:    sum %= tall  (sum=sum % tall)

 

 

Logiske

&&    (AND) Sjekker om flere verdier tilfredsstiller et krav

||    (OR) Sjekker om en eller flere verdier tilfredsstiller et krav

!    (NOT) Sjekker om en eller flere verdier ikke tilfredsstiller et krav

 

Eksempel

            Console.WriteLine("Multiplikasjon: " + multiplikasjon);

            // Tilordne

                Console.WriteLine("AND:  None of the numbers is larger than 10");

            }

                Console.WriteLine("NOT:  No number is larger than 10");

                Console.WriteLine("NOT:  Not all of the numbers is larger than 10");

I C# brukes for løkker til å utføre et sett med kommandoer et bestemt antall ganger.  Antall ganger kan enten være et tall eller en variabel.

·         Kommandoen starter med for: for (int teller = 1; teller <= 10; teller++)

·         Etter for er det en parentes i motsetning til Python:  for (int teller = 1; teller <= 10; teller++)

·         Inne i parentesen er det tre elementer.

·         Det første elementet angir en startverdi for den variabelen som skal holde orden på hvor mange runder løkken har gått: for (int teller = 1; teller <= 10; teller++)

·         Den andre elementet er betingelsen. I eksemplet er betingelsen så lenge teller er mindre eller lik 10: for (int teller = 1; teller <= 10; teller++)

·         Det tredje elementet er å øke teller med 1: for (int teller = 1; teller <= 10; teller++)

Classes og Objects

Class

En class er en beskrivelse av de egenskapene som gjør at et objekt er det det er.

 

Objekt

Når vi tenker på objekt, teker vi ofte på noe fysisk som vi kan se eller ta på.

 

Instance

Et objekt som er basert på en klasse

 

Class

En class er en beskrivelse av hva et objekt skal bestå av.

 

I C# lager du en class ved å bruke class

 

Eksempel :

using System;

 

class Program

{

    class Datamaskin // Lager en class

    {

        public string type; // Lager et field i class for å holde en tekststreng

        public double pris; // Lager et field i class for å holde en double

    }

    static void Main()

    {

        Datamaskin maskin1 = new Datamaskin();  // Lager en instance av class datamaskin

        maskin1.type = "Spill PC";              // tilordner verdi til type i instance maskin1

        maskin1.pris = 23999.99;                // tilordner verdi til pris i instance maskin1

        Datamaskin maskin2 = new Datamaskin();  // Lager en instance av class datamaskin

        maskin2.type = "Kontor PC";             // tilordner verdi til type i instance maskin2

        maskin2.pris = 3999.99;                 // tilordner verdi til pris i instance maskin2

 

// Skriver ut tekst og to field fra første instance

Console.Write("Maskin nummer 1: " + maskin1.type +"koster "+ maskin1.pris+"\n");   

// Skriver ut tekst og to field fra andre instance

Console.Write("Maskin nummer 1: " + maskin2.type +"koster "+ maskin2.pris);            

}

}

 

 

I eksempelet over lager vi class datamaskin ved å skrive class Datamaskin

 

    class Datamaskin // Lager en class

    {

        public string type; // Lager et field i class for å holde en tekststreng

        public double pris; // Lager et field i class for å holde en double

    }

 

I class legger jeg type og pris som variabler i class. En variabel som opprettes i en class kalles field (eller attribute).

 

Det er ikke nødvendig, men anbefalt å starte navn på class med en stor forbokstav. Det er også vanlig å gi C# filen samme navn som class. Det er ikke nødvendig i motsetning til noen andre språk.

 

Hvordan lage et Object

Et object blir laget fra beskrivelsene i en class.

 

Datamaskin maskin1 = new Datamaskin();  // Lager en instance av class datamaskin

 

For å lage et Objekt av typen Datamaskin, skriver du class navnet, et = tegn og keyword new og navnet du vil object skal ha.  Til slutt skriver du class med parenteser.

 

 

Sende med arguments i opprettelsen av Object.

I eksempel under sender vi med type og pris i opprettelsen av objektet og sparer oss derfor å bruke kode for å legge inn data i objektene etterpå.  Legg merke til at jeg bruker this før fields. Det betyr at jeg kobler field til constructor.

Variabler bør være av typen private. Det betyr at bruken av de bare kan brukes i det namespace de står i. Dette er den mest ryddige måten å programmere på, og veldig anbefalt.

 

Eksempel :

using System;

 

class Datamaskin

{

    private string type;

    private double pris;

 

    public Datamaskin (string type, double pris)

    {

        this.type = type;

        this.pris = pris;

    }

    static void Main()

    {

        Datamaskin maskin1 = new Datamaskin("Spill PC", 23999.99);

        Datamaskin maskin2 = new Datamaskin("Kontor PC", 3999.99);

 

        Console.Write("Maskin nummer 1: " + maskin1.type +"koster "+ maskin1.pris+"\n");

        Console.Write("Maskin nummer 1: " + maskin2.type +"koster "+ maskin2.pris);

    }

}

 

 

Ulempen er jo da at vi må endre navnet på programmet til class navn. Class Datamaskin gjør vi om til en constructor, noe som igjen gjør at main funksjonen havner inne i en annen funksjon, og ikke blir stående som en egen funksjon. Dette er ikke bra programmering.

Bruke flere Class

For å flytte ut en class i en egen fil mår du først lage en ny cs fil

 

Så må vi starte med å flytte class til det nye dokumentet. I og med at variablene er private, noe som gjør at vi må vi lage properties som gjør at vi kan bruke fields .

 

public string Type { get { return type; } set { type = value; } }

public double Pris { get { return pris; } set { pris = value; } }

 

Legg merke til at properties skrives med stor forbokstav tilsvarende field de representerer.

Legg merke til at jeg bruker using namespace fra class filen i programfilen min.

 

Fil Datamaskin.cs

using System;

 

namespace enhet.Datamaskiner

{

    public class Datamaskin

    {

        private string type;

        private double pris;

        public string Type { get { return type; } set { type = value; } }

        public double Pris { get { return pris; } set { pris = value; } }

 

 

        public Datamaskin(string type, double pris)

        {

            this.Type = type;

            this.Pris = pris;

        }

     }

}

    }
